# Sales Analysis Dashboard

## Project Overview

This project analyzes sales data using Python, MySQL, Pandas, Matplotlib, and Power BI to generate business insights and visualize sales performance.

The project extracts data from a MySQL database, performs data cleaning and transformation, calculates key performance indicators (KPIs), and creates interactive visualizations to support business decision-making.

## Data Processing

The following data preparation steps were performed:

- Extracting data from MySQL using SQL JOINs
- Loading data into Pandas DataFrames
- Removing duplicate records
- Handling missing values
- Creating calculated revenue columns

Revenue Formula:

```python
Revenue = Price × Quantity
```

---

## SQL Query Used

```sql
SELECT
    c.customer_name,
    p.product_name,
    s.quantity,
    p.price
FROM customers c
JOIN sales s
ON c.customer_id = s.customer_id
JOIN products p
ON p.product_id = s.product_id;
```
